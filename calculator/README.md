# AI Agent

This is an AI agent built for the purpose to function a helper in a carefully built linux subsystem. It is not for public use, but is a working project for learning how AI Agents function. Built with the help of the learning course "Build an AI Agent" from boot.dev from all commits up to "Finihsed CH4:Agents". 

# Extending the Project
Ways to extend/improve the project:

Fix harder and more complex bugs
Refactor sections of code
Add entirely new features
You can also try:

Other LLM providers
Other OpenRouter models (try pinning a specific :free model instead of openrouter/free)
Giving it more functions to call
Other codebases (commit your changes before running the agent on a codebase, so you can always revert)
